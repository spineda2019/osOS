osOS Documentation

Welcome! This is the landing page for the documentation of my osOS kernel pet project.

The docs themselves are generated by the zig build system, this page is just a landing page to glue them all together in one neat home since there is architecture specific code for each architecture's entry point. Here they are:

  • x86 Kernel Documentation Home Page
  • RISC-V32 Kernel Documentation Home Page

Of course, if you would like to check out the source, you can check out the repo hosted on github:

  • Source Repo

Github stars are appreciated! But of course, they are not required.
